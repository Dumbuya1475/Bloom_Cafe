# Bloom Café Landing Page ☕

A modern, responsive landing page built for a fictional coffee shop as part of a freelance-style bounty project under the Learn2Earn Program (Cohort 2).

This project simulates real-world client work, focusing on clean UI design, responsiveness, and component-based architecture using modern frontend tools.

## 📌 Project Overview

Bloom Café is a fictional coffee shop website designed to attract customers and showcase menu offerings, services, and contact details in a visually appealing and user-friendly way.

The goal of this project is to demonstrate the ability to:

* Interpret a client brief
* Build a structured, responsive UI
* Deliver a production-ready frontend project

---

## 🛠️ Tech Stack

* React (Vite)
* Tailwind CSS
* React Icons

---

## 🎯 Features

* Fully responsive design (mobile, tablet, desktop)
* Sticky navigation with smooth scrolling
* Hero section with clear call-to-action
* About section with brand story
* Featured menu showcase
* Service highlights section
* Customer testimonials
* Contact form UI
* Footer with social links

## 🎓 Context

This project was completed as part of the Learn2Earn Program (Cohort 2), an initiative by the Ministry of Communication, Technology and Innovation (MoCTI), UNICEF Sierra Leone, and partners.

The program focuses on preparing young people for freelance and digital work opportunities through practical, real-world assignments.
